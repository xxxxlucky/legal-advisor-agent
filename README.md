# ⚖️ 智能法律顾问助手 Agent

基于 Dify Chatflow 搭建的 **Workflow-based Agent**，面向劳动法场景，提供「AI 法律咨询 + 律师智能路由」的一站式服务。覆盖 15 部核心法律法规，支持意图识别、法规检索增强问答、案情紧急度评估和律师自动分配。

---

## 🎯 核心能力

| 能力 | 说明 |
|---|---|
| 智能意图识别 | 三态分流：法律咨询 / 超出范围 / 简短确认，非法律问题自动拦截 |
| RAG 增强问答 | 混合检索（语义 0.5 / 关键词 0.5）+ Rerank，确保法条引用准确 |
| 四段式回复 | 情绪安抚 → 法律定性 → 实操步骤 → 风险提示，专业且克制 |
| 律师智能路由 | 按「生存威胁优先」原则自动分配（张三/李四/王五/需用户确认） |
| 双护栏安全 | 业务边界拦截（IF1）+ 安全底线检测（IF2） |
| 降级兜底 | JSON 解析失败 / 模型超时 / 检索为空 → 均有预设策略 |

---

## 🏗️ 架构

```
用户输入
  ↓
LLM0（意图识别）
  ↓
IF1（护栏1：业务边界）── 非法律 → "不在范围内" → 结束
  ↓ 是法律问题
知识库检索（混合检索 + Rerank）
  ↓
LLM1（咨询：情绪安抚→定性→步骤→风险）
  ↓
Code2（JSON解析 + 安全检测）
  ↓
IF2（护栏2：安全底线）── 危险内容 → "无法提供" → 结束
  ↓ 安全
IF3（need_lawyer判断）
  ├─ false → 直接输出咨询回复
  └─ true  → LLM2（律师路由）→ 合并输出咨询 + 推荐
```

## 🧪 实测结果

在 8 组典型劳动纠纷 case 上验证：

| Case | 类型 | 路由结果 | 是否正确 |
|---|---|---|---|
| 试用期被辞退不给赔偿 | 辞退纠纷 | 未触发律师（轻咨询） | ✅ |
| 年假怎么算 | 法律咨询 | 未触发律师（常识问询） | ✅ |
| 工地摔伤 公司不认工伤 花两万多 | 工伤 + 公司拒赔 | → 王五 | ✅ |
| 工厂砸断脚 老板不管还辞退 | 工伤 + 辞退 | → 王五（备选张三） | ✅ |
| 拖欠工资 + 孕期被调岗 | 欠薪 + 女职工 | → 张三（备选王五） | ✅ |
| 今天天气怎么样 | 非法律 | 护栏拦截 | ✅ |
| 帮我写 Python 爬虫 | 非法律 | 护栏拦截 | ✅ |
| *(1 个边界 case)* | 工伤 + 公司拒赔 | need_lawyer 漏判 | ⚠️ 已修复 |

## 🔧 快速复现

1. 注册 [Dify Cloud](https://cloud.dify.ai) 或本地部署
2. 创建 Chatflow → 导入本项目的配置
3. 上传 `labor-laws.md` 至知识库
4. 配置 DeepSeek API Key（模型供应商页）
5. 按 [dify-config.md](./dify-config.md) 逐节点搭建
6. 发布 → 测试
